# secure_coding_review
This project focuses on Secure Coding Review, where a sample program is analyzed to identify security vulnerabilities and demonstrate how they can be fixed using secure coding practices.  In this task, a vulnerable login system is reviewed to show how improper handling of user input can lead to SQL Injection attacks.
